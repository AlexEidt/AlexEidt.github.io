Projects


UW Course Planner

The UW Course Planner is a Web App with schedule visualization on a campus map, recursive course prerequisite tree generation and course keyword search for courses offered at the University of Washington. I also built a library called uwtools to parse course data from the UW course catalog and time schedule.

UIL/UW Solar Websites

I created and designed the new website for UW Solar and its parent lab, the Urban Infrastructure Lab (UIL) along with a fellow Computer Science undergraduate. Our goal was to make the site simple and robust so future students could edit and understand it easily.

Basketball Statistics Tracking

To keep track of player statistics for my Intramural Basketball Team, I built an application in Java. I used Java Swing to create an intuitive UI for creating teams and adding statistics.

Word Search Generator

A simple Word Search Generator that allows the user to show the solution on the board, reshuffle the current words, or create a new board with new words.
