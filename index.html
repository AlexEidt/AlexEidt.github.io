Projects


UW Course Planner

The UW Course Planner is a Web App with schedule visualization on a campus map, recursive course prerequisite tree generation and course keyword search for courses offered at all campuses of the University of Washington. The backend of the website is built with Python using the Flask framework. I also built an accompanying library called uwtools to parse course data from the UW course catalog and time schedule websites.

Check out the project on GitHub!
Check out uwtools on GitHub!

Urban Infrastructure Lab and UW Solar Websites

While part of the UW Solar Registered Student Organization at the University of Washington, I created and designed the new UW Solar website along with a fellow Computer Science undergraduate. Our main goal was to make the site simple and robust such so that future UW Solar students could edit and understand it easily. To help with this, I created documentation for the site. I also designed and created the Urban Infrastructure Lab website using the UW WordPress template and got it hosted with the help of two UW Web developers.

Check out the UW Solar site on GitHub!

Basketball Statistics Tracking

Word Search Generator

A simple Word Search Generator in Python. The user can enter their own words and customize their board size or go let the program randomly choose words for them. The app allows the user to show the solution on the board, reshuffle the current words, or create a new board with new words.

Check out the project on GitHub!
